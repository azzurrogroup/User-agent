# User-agent

Zgodnie z komunikacja developerska dot. user agent poniżej przedstawiam opis w jakim celu jest wykorzystywany token API.

Programy/skrypty korzystające z tokena mają za zadanie w sposób inwidualny stworzony pode mnie jako użytkownika API:
- generować automatycznie etykiety
- zwiększać efektywność komunikacji
- prowadzić masowe wgrywanie dokumentów
- informować o bieżących sprawach z inwidualnym ustaleniem priorytetu
- sprawnie konfigurować aukcje
- oraz inne pomniejsze sprawy bieżące

Testowa wersja web odnosi się do tych samych skryptów natomiast testowanych w architekturze webowej - bardziej dostepnej.
